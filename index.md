# 欢迎使用 Horizon

这是一个由 AI 驱动的信息聚合系统。它会自动从 GitHub、Hacker News、RSS、Reddit 等渠道抓取高价值信息，并为您生成每日摘要。

### 📌 如何使用
1. **点击左侧边栏**：查看不同日期的每日速递。
2. **使用右侧目录**：在长篇日报中快速定位感兴趣的内容。
3. **查看文档**：了解配置和评分机制。
